# Roadmap – TaxiMaps

## MVP (0–6 weeks)
- Define P0 utilities and dashboards
- Add event tracking schema (core events)
- Release MVP to a small test group
- Evaluate adoption + retention

## V1 (6–12 weeks)
- Expand utility set based on usage data
- Saved presets + preferences
- Improve alert rules and reliability

## V2 (12+ weeks)
- Personalisation (based on usage)
- Deeper analytics and insights
- More integrations (where valuable)
